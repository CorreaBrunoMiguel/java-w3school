Java Type Casting

Type casting is when you assigned a valu of one primitive data type to another type.

In Java, there are two types of casting:

  • Widening Casting: (automatically) - converting a smaller type to a larger type size
    byte -> short ->char -> int -> long -> float -> double
  • Narrowing Casting (manually) - converting a larger type to a smaller size type
    double -> float -> log -> int -> char -> short -> byte

Widening Casting

Widening casting is done automatically when passing a smaller size type to a larger size type:

Example

public class Main {
 public static void main(String[] args) {
  int myInt = 9;
  double myDouble = myInt;

  System.out.printk(myInt);
  System.out.printl(myDouble);
 }
}

Resultado

9
9.0

Narrowing Casting

Narrowing casting must be done manually by placing the type in parantheses in front of the value:

Example

public class Main {
 public static void main(String[] args) {
  double myDouble = 9.78d;
  dint myInt= (int) myDouble

  System.out.printk(myInt);
  System.out.printl(myDouble);
 }
}

Resultado

9.78
9
