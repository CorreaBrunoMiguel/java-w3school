Java Comments

Java Comments

Comment can be used to explain Java code, and to make it more readable. It can also be used to prevent execution when testing alternative code.

Example

// This is a comment

System.out.println("Hello World")

Output

Hello World!

This example uses a single-line comment at the end of a line code:

Example

System.out.println("Hello World") // This is a comment

Java Multi-line Comments

Multi-line comments start with /* and ends with */. Any text between will be ignored by Java.

Example

/* The code below will print the words Hello WOrld to the screen, and it is amazing */

System.out.println("Hello World")
