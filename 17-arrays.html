Java Arrays

Java Arrays

Arrays are used to store multiple values in a single variable, instead of declaring separate variables for each value.

To declare an array, define the variable type with square brackets:

String[] cars;

We have now declared a variable that holds an arrays of strings. To insert values to it, you can place the values in a comma-separated list, inside curly braves:

String[] cars = {"Volvo", "BMW", "Ford"};

To creare an array of integers, you could write:

int[] myNums = {10, 20, 30, 40};

Access the Elements of an Array

You can access an array element by referering to the index number.

Example

String[] cars = {"Volvo", "BMW", 'Ford', "Mazda", "Fiat"};
System.out.println(cars[0]);

Result

Volvo

Change an Array Element

To change the value of a specific element, refer to the index number:

Example

String[] cars = {"Volvo", "BMW", 'Ford', "Mazda", "Fiat"};
cars[0] = "Nissan";
System.out.println(cars[0]);

Result

Nissan

Array Length

To find out how many elements an array has, use the length property:

Example

String[] cars = {"Volvo", "BMW", 'Ford', "Mazda", "Fiat"};
System.out.println(cars.length);

Result

5

Java Arrays Loop

Loop Trough an Array

You can loop trough the array elements with the for loop, and use the length property to specify how many times the loop should run.

Example

String[] cars = {"Volvo", "BMW", 'Ford', "Mazda", "Fiat"};
System.out.println(cars.length);

Result

5
