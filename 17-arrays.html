Java Arrays

Java Arrays

Arrays are used to store multiple values in a single variable, instead of declaring separate variables for each value.

To declare an array, define the variable type with square brackets:

String[] cars;

We have now declared a variable that holds an arrays of strings. To insert values to it, you can place the values in a comma-separated list, inside curly braves:

String[] cars = {"Volvo", "BMW", "Ford"};

To creare an array of integers, you could write:

int[] myNums = {10, 20, 30, 40};

Access the Elements of an Array

You can access an array element by referering to the index number.

Example

String[] cars = {"Volvo", "BMW", 'Ford', "Mazda", "Fiat"};
System.out.println(cars[0]);

Result

Volvo

Change an Array Element

To change the value of a specific element, refer to the index number:

Example

String[] cars = {"Volvo", "BMW", 'Ford', "Mazda", "Fiat"};
cars[0] = "Nissan";
System.out.println(cars[0]);

Result

Nissan

Array Length

To find out how many elements an array has, use the length property:

Example

String[] cars = {"Volvo", "BMW", 'Ford', "Mazda", "Fiat"};
System.out.println(cars.length);

Result

5

Java Arrays Loop

Loop Trough an Array

You can loop trough the array elements with the for loop, and use the length property to specify how many times the loop should run.

Example

String[] cars = {"Volvo", "BMW", 'Ford', "Mazda", "Fiat"};

for (int i = 0; i < cars.length; i++) {
 system.out.println(cars[i]);
}

Result

Volvo
BMW
Ford
Mazda
Fiat

Loop Trough an Array with For-Each

There is also a for-each loop. which is used exclusively to loop trough elements in arrays:

Example

String[] cars = {"Volvo", "BMW", 'Ford', "Mazda", "Fiat"};

for (String i : cars) {
 system.out.println(cars[i]);
}

Result

Volvo
BMW
Ford
Mazda
Fiat

The example above can be read like this: for each String element (called i - as in index) in cars, print out the value of i.

If you compare the for loop and for-each loop, you will see that the for-each method is easier to write, it does not require a counter (using the length property), and it is more readable.

Java Multi-Dimensional Arrays

Multidimensional Arrays

A multidimensional array is an array of arrays.

Multidimensional array are useful when you want to store data as a tabular form, like table with rows and columns.

To create a two-dimensional array, add each array within its own set of curly braces:

Example

int[][] myNumbers = {{1, 2, 3, 4}, {5, 6, 7}};

myNumbers is now an array with two arrays as its elements.

Access Elements

To access the elements of the myNumbers array, specify two indexes: one for the array, and one for the element inside that array. This example accesses the third element (2) in the second array (1) of myNumbers:

Example

int[][] myNumbers = {{1, 2, 3, 4}, {5, 6, 7}};
System.out.println(myNumbers[1][2]);

Result

7

Change Elements Values

You can also change the value of an element:

Example

int[][] myNumbers = {{1, 2, 3, 4}, {5, 6, 7}};
myNumber[1][2] = 9; System.out.println(myNumbers[1][2]);

Result

9

Loop Trough a Multi-Dimensional Array

We can also use for loop inside another for loop to get the element of a two-dimensional array (we still have to point to the two indexes):

Example

public class Main {
 public static void main (String[] args) {
  int[][] myNumbers = {{1, 2, 3, 4}, {5, 6, 7}};
  for (int i = 0; i < myNumbers.length; ++i) {
   for (int j = 0; j < myNumbers[i].length; ++j) {
    System.out.println(myNumbers[i][j]);
   }
  }
 }
}
