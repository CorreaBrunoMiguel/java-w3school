Java Break and Continue

Java Break

You have already seen the break statement used in an earlier chapter of this tutorial. It was used to "jump out" of a switch statement.

The break statement can also be used to jump out a loop.

Example

for (int i = 0; i < 10; i++) {
 if (i == 4) {
  break;
 }
 System.out.println(i);
}

Result

0
1
2
3

Java Continue

The continue statement breaks one iteration (in the loop), is a specified condition occurs, and continue with the next iteration in the loop.

Example

for (int i = 0; i <= 5; i++) {
 if (i == 3) {
  continue;
 }
 System.out.println(i);
}

Result

0
1
2
3
5

Break and Continue in While Loop

You can also use break and continue in while loops:

Break Example

int i = 0;

while (i < 6) {
 System.out.println(i);
 i++;
 if (i == 3) {
  break;
 }
}

Result

0
1
2
3

Continue Example

int i = 0;

while (i < 6) {
 if (i == 3) {
  i++;
  continue;
 }
 System.out.println(i);
 i++;
}

Result

0
1
2
4
5
